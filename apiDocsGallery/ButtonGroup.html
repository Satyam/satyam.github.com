Class ButtonGroup - extends Y.Widget - uses Y.WidgetParent, Y.Makenode

The ButtonGroup class provides a container for sets of Buttons. All buttons added will be extended with Y.WidgetChild.

Constructor

ButtonGroup ( cfg )
Parameters:
cfg <object> Configuration attributes

Properties

BOUNDING_TEMPLATE - String

Overrides the standard bounding box template to produce a <fieldset>

Y.ButtonGroup._ATTRS_2_UI - protected static Object

Hooks up _uiSetLabel to respond to changes in the label attribute.

Y.ButtonGroup._CLASS_NAMES - protected static [Strings]

Creates the label className key to be used in the template

Y.ButtonGroup._TEMPLATE - protected static String

Template for the <legend> element to hold the label. Used by MakeNode.

Configuration Attributes

alwaysSelected - Boolean

Forces this group to always have at least one toggle button selected
Default Value: false

defaultChildType - object

Defines the default type of child to be contained in this group. Used by WidgetParent to create the default children
Default Value: Y.Button

label - String

Holds the label for this group of buttons
Default Value: ""
