gallery-flyweight-tree Module

Defined in: yui3-gallery/src/gallery-flyweight-tree/js/flyweight-tree-node.js:12

This module provides the following classes:

  • Y.FlyweightTreeManager
  • Y.FlyweightTreeNode

This module is a rollup of the following modules:

  • flyweight-tree-manager
    Extension to handle its child nodes by using the flyweight pattern.
  • flyweight-tree-tnode
    An implementation of the flyweight pattern. This object can be slid on top of a literal object containing the definition of a tree and will take its state from that node it is slid upon. It relies for most of its functionality on the flyweight manager object, which contains most of the code.
