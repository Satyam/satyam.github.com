gallery-button-plugin Module

Defined in: src/gallery-button-plugin/js/button-plugin.js:11

Node plugin to handle toggle buttons and groups of mutually exclusive toggle buttons. Searches a given container for buttons marked with the yui3-button-toggle className and turns them into toggle buttons and also any HTML element with the yui3-button-group-exclusive className and makes the toggle buttons within it mutually exclussive. Adds the selected attribute, for toggle buttons it tells whether the button is in the pressed state, for groups of toggles points to the button currently presssed. Relies on the cssbutton module for styling.
