Class TimeSpinner - extends Widget - uses Y.Spinner, Y.MakeNode

The TimeSpinner shows a set of spinners for hour, minute and optionally seconds and AM/PM.

Constructor

TimeSpinner ( cfg )
Parameters:
cfg <object> (optional) configuration attributes

Properties

_ampmSp - private Y.Spinner

Reference to the Spinner showingam/pm.
Default Value: null

_frozenTime - private Number (milliseconds)

Used to store the moment when the timer was stopped
Default Value: 0

_hourSp - private Y.Spinner

Reference to the Spinner showing the hours.
Default Value: null

_minSp - private Y.Spinner

Reference to the Spinner showing the minutes.
Default Value: null

_offset - private Number

Difference in milliseconds in between the current time and that displayed. Used when the timer is running to display times other than the current time.
Default Value: 0

_secSp - private Y.Spinner

Reference to the Spinner showing the seconds.
Default Value: null

_timer - private timer object

Reference to the timer created by Y.later that updates the time.
Default Value: null

Configuration Attributes

interval - Number (milliseconds)

How often to refresh the time displayed
Default Value: 500

running - Boolean

Whether the timer is running.
Default Value: true

showAmPm - Boolean

Whether to show the am/pm indicator or show a 24 hours timer.
Default Value: false

showSeconds - Boolean

Whether to show the seconds indicator
Default Value: true

strings - object

Set of localizable strings to be used as tooltips on the spinners.

value - Date

Value of the timer
Default Value: time at initialization or current time if running
