find

(
  • value
  • [move]
)
Integer | null

Defined in yui3-gallery/src/gallery-md-model/js/gallery-md-model.js:1497

Locates a record by value. The record will be located by the field given in the sortField attribute. It will return the index of the record in the shelves or null if not found. By default it will expose that record. If sortField contains a function, it will return null and do nothing. Since sort fields need not be unique, find may return any of the records with the same value for that field.

Parameters:

  • value Any

    value to be found

  • [move] Boolean optional

    exposes the record found, defaults to true

Returns:

Integer | null: index of the record found or null if not found. Be sure to differentiate a return of 0, a valid index, from null, a failed search.

sortField

String | Function

Defined in yui3-gallery/src/gallery-md-model/js/gallery-md-model.js:1532

Name of the field to sort by or function to build the value used for comparisson. If a function, it will receive a reference to the record to be sorted; it should return the value to be used for comparisson. Functions are used when sorting on multiple keys, which the function should return concatenated, or when any of the fields needs some pre-processing.

Default: first primary key field
