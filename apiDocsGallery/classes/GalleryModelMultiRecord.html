GalleryModelMultiRecord Class

Defined in: src/gallery-md-model/js/multi-record.js:1
Module: gallery-md-model

Allows GalleryModel to handle a set of records using the Flyweight pattern. It exposes one record at a time from a shelf of records. Exposed records can be selected by setting the GalleryModel/index:attribute attribute.

each

(
  • fn
)
chainable

Defined in src/gallery-md-model/js/gallery-md-model.js:1086

Executes the given function for each record in the set. The function will run in the scope of the model so it can use this.getValue() or any such method to access the values of the current record. Returning exactly false from the function spares shelving the record. If the callback function does not modify the record, returning false will improve performance.

Parameters:

  • fn Function

    function to execute, it will be provided with:

    • index Integer

      index of the record exposed
