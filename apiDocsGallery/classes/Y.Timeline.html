Properties

_events

Object

Defined in yui3-gallery/src/gallery-timeline/js/gallery-timeline.js:47

Stores the events to display.It contains the following properties, those starting with underscore are used internally:

  • start: {timestamp} start time in milliseconds
  • end: {timestamp} end time in milliseconds
  • text: {string} text to be shown on the bar
  • fuzzy: {Boolean} the start and end days are uncertain
  • locked: {Boolean} the event cannot be edited (not relevant for this viewer
  • endsToday: {Boolean} the end day is today
  • description: {string} extended description
  • icon: {string} Base64-encoded image to go along the extended description
  • category: {string} the category this event belongs to
  • bar: {Y.Node} reference to the Node for the bar representing this event
  • pointer: {Y.Node} for point events, reference to the Node for the date pointer
  • _isPoint: {Boolean} signals that the event is a point event or a range event that has become too narrow to be displayed as a range

Default: null

_mode

integer

Defined in yui3-gallery/src/gallery-timeline/js/gallery-timeline.js:67

Display mode for the bars, to help calculate its location and handle crowding. Can be standard (0), , compact (1) or overlappying (2)

Default: 0

Attributes

categories

Object

Defined in yui3-gallery/src/gallery-timeline/js/gallery-timeline.js:722

Stores the categories, indexed by category name. Each category contains:

  • color: {string} Background color for the bar in #rrggbb format
  • fontColor: {string} Color for the text in the bar in #rrggbb format
  • hidden: {Boolean} Events in this category should not be shown

Default: {}
