selected

Integer

Defined in yui3-gallery/src/gallery-fwt-treeview/js/fw-treenode.js:162

Selected/highlighted state of the node. It can be

  • Y.FWTreeNode.NOT_SELECTED (0) not selected
  • Y.FWTreeNode.PARTIALLY_SELECTED (1) partially selected: some children are selected, some not or partially selected.
  • Y.FWTreeNode.FULLY_SELECTED (2) fully selected.

The partially selected state can only be the result of selection propagating up from a child node. The attribute might return PARTIALLY_SELECTED but the developer should never set that value.
