Class FieldDefSet

Collection of FieldDef instances

Constructor

FieldDefSet ( defs , _parent )
Parameters:
defs <FieldDef or object> Initial set of definitions
_parent <FieldDef> (private) Reference to the parent definition of this collection

Properties

_items - private [strings]

Array containing the names of the fields in this level. It is this array that gives the order of the fields when traversing them
Default Value: []

_keys - private object

Collection of FieldDef indexed by name. Valid only on the root FieldDefSet.
Default Value: {}

_parent - private FieldDef

Reference to the FieldDef instance this set is a child of
Default Value: null

_root - private FieldDefs

Reference to the top-most FieldDefSet instance
Default Value: null

_totalDepth - private integer

Number of levels of nesting. Valid only on the root FieldDefSet.
Default Value: 0

FieldDefSet.TRAVERSE_DEEP - static string

Constant to indicate traversal method

FieldDefSet.TRAVERSE_LEAVES - static string

Constant to indicate traversal method

FieldDefSet.TRAVERSE_SHALLOW - static string

Constant to indicate traversal method

traverse

void traverse ( method , fn , context , _depth )
Traverses the field definitions tree in several ways, calling the provided function for each field definition. Traversing is done in the order in which the definitions were added, unless added with explicit indices.
Parameters:
method <string> Any of the following constants:
  • FieldDefSet.TRAVERSE_SHALLOW fieldDefs are traversed by level, useful for building HTML table headers
  • FieldDefSet.TRAVERSE_DEEP fieldDefs are traversed going deep into each branch before going to the next, useful for building HTML forms with nested fieldSets.
  • FieldDefSet.TRAVERSE_LEAVES only fieldDefs which have no children, lists only the ones that store data
fn <function> Function to be called for each field definition. It will receive:
  • fieldDef Definition for each field
  • depth Level of nesting of this definition
context <object> (optional) Context to execute the function in (defaults to this FieldDefSet instance)
_depth <integer> (private) Used to keep track of the depth in recursive calling.
Returns: void
