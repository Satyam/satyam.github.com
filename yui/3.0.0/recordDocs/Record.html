Class Record - extends Base

The Record class provides storage for a single row of data.

Parameters:

Methods

accept

void accept ( fieldName )
Accepts the current value of a field or of whole record as good. Used when the Record has been synchronized with the original source (usually a database table in the server) to signal that the current value has been accepted as good by the server.
Parameters:
fieldName <string> (optional) If provided, it will accept the named field, otherwise, it accepts the whole record
Returns: void

Returns the value of a field or an object with all the field values

Restores the initial value of a field or of a whole record

fieldName <string> (optional) If provided, it will reset the named field, otherwise, it resets the whole record

Sets the value for a specific field. If field does not exist, it creates it.
Parameters:
fieldName <string> Name of the field

undo

void undo ( fieldName )
Restores the previous value of a field or of a whole record. There is only one level of undo.
Parameters:
fieldName <string> (optional) If provided, it will undo the named field, otherwise, it undoes the whole record
Returns: void

Configuration Attributes

deleted - Boolean

Marks the record as deleted

dirty - Boolean

True when any of the fields is dirty. Read only.

new - Boolean

True when all the fields are new. Read Only
