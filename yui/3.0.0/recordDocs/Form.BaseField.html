Class Form.BaseField - extends Widget

Base for form input fields. It displays a textBox.

Properties

_input - private HTMLElement

Holds a reference to the actual input element

_span - private HTMLElement

Holds a reference to the SPAN element that shows the value of the field

FIELD_TEMPLATE - static string

Template to build the label and input elements with. The template will have the classNames substituted in first and then the values
