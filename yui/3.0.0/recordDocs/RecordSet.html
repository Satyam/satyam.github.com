Hash containing the list of serializers available. The developer is expected to add to this list, if so wished. The following serializers are already set:

Configuration Attributes

dataSource - Y.DataSource

Instance of DataSource to use for loading the RecordSet
Default Value: null

fieldDefSet - FieldDefSet

Holds a reference to the FieldDefSet instance which defines the properties of the Records stored

serializeDirtyOnly - Boolean

Signals that only dirty fields should be included in the serialization
Default Value: true

serializeFormat - string or function

Format of the RecordSet serialization. Can be any of the properties in RecordSet.Serializers (built in or added by the developer) or a function that will receive an object to serialize and should return a string
Default Value: "JSON"

sortCompareFunction - function or null

Function to sort records by. If none provided (the default) regular comparisons will be used. The function will receive:
  • a: first record to compare
  • b: second record to compare
  • desc: Boolean indicating the sort should be in descending order
  • fieldName: Name of the field to sort by
    • Default Value: null.

    sortedBy - object

    Signals if the RecordSet is sorted, on which field and in which direction. It contains:
    • fieldName: Name of the field it is sorted by
    • desc: true if sort was descending
    Default Value: null (not sorted)
