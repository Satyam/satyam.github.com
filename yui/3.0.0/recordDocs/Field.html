Class Field

The Field class stores a single piece of data, keeping track of its modified (dirty) or newly created(new) current, previous and initial values.

Constructor

Field ( value )
Parameters:
value <mixed> initial value for this field

Properties

_dirty - private Boolean

Signals if the value was modified since last created or accepted as good

_initialValue - private any

Stores the initial value of the field

_new - private Boolean

Signals if the field has been newly created instead of loaded from a remote source

_oldValue - private any

Stores the most recent previous value, useful for undoing

value - any

Stores the value for this field
