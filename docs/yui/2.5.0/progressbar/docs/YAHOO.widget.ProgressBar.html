Class YAHOO.widget.ProgressBar - extends YAHOO.util.Element

A Progress Bar providing various visual options, animation, it can grow vertically or horizontally in any direction and fires several events while moving.

Constructor

YAHOO.widget.ProgressBar ( oConfigs )
Parameters:
oConfigs <object> An object containing any configuration attributes to be set

Methods

destroy

destroy ( )
It will destroy the ProgressBar, related objects and unsubscribe all events

render

YAHOO.widget.ProgressBar render ( el , before )
It will render the ProgressBar into the given container. If the container has other content, the ProgressBar will be appended to them. If the second argument is provided, the ProgressBar will be inserted before the given child. The method is chainable since it returns a reference to this instance.
Parameters:
el <HTML Element> HTML element that will contain the ProgressBar
before <HTML Element> (optional) If present, the ProgressBar will be inserted before this element.
Returns: YAHOO.widget.ProgressBar
Chainable: This method is chainable.

Configuration Attributes

anim - {instance of YAHOO.util.Anim}

A reference to the YAHOO.util.Anim instance attached to the bar. (ReadOnly) If the Animation utility is loaded, it will be automatically used.
Default Value: null

ariaText - String

The text to be voiced by screen readers. The text should contain a vertical bar character where the current value of the bar should be inserted.
Default Value: '|'

backColor - String - CSS color specification

Color for the background. It can be any valid CSS color value.
Default Value: 'white'

barColor - String - CSS color specification

Color for the bar. It can be any valid CSS color value.
Default Value: 'blue'

barEl - HTMLElement (div)

Reference to the HTML object that makes the moving bar (read-only)

border - String - CSS border specification

CSS attributes for the border.
Default Value: 'none'

direction - String

Direction of movement of the bar. It can be any of 'lr' (left to right), 'rl' (the reverse) , 'tb' (top to bottom) or 'bt', case-insensitive. Can only be set once and only before rendering.
Default Value: 'lr'

height - Number or String

Height of the ProgressBar. If a number, it will be presumed to be in pixels. If a string it should be a valid setting for the CSS height attribute. It will always be returned as a string including units.
Default Value: 20px

maskEl - HTMLElement (table)

Reference to the HTML object that overlays the bar providing the mask. (read-only)

maxValue - Number

Represents the top value for the bar. The bar will be fully extended when reaching this value. Values higher than this will be ignored.
Default Value: 100

minValue - Number

Represents the lowest value for the bar. The bar will be totally collapsed when reaching this value. Values lower than this will be ignored.
Default Value: 0

value - Number

The value for the bar. Valid values are in between the minValue and maxValue attributes.
Default Value: 50

width - Number or String

Width of the ProgressBar. If a number, it will be presumed to be in pixels. If a string it should be a valid setting for the CSS width attribute. It will always be returned as a string including units.
Default Value: 200px
